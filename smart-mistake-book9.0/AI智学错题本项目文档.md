# AI智学错题本项目文档

## 1. 项目概述

AI智学错题本是一款融合AI技术与科学学习方法的智能错题管理系统，旨在解决传统错题本记录效率低、分析深度浅、复习不科学等问题。系统采用本地大模型分析、OCR识别、艾宾浩斯复习引擎等核心技术，为中小学生提供高效、智能的错题管理解决方案。

## 2. 痛点问题分析

### 2.1 传统错题本的核心痛点

| 痛点 | 具体表现 | 影响 |
|-----|---------|------|
| **记录效率低** | 手写抄题耗时费力，尤其数学题、图表题难以复制 | 学生不愿坚持记录，错题积累不足 |
| **分析深度浅** | 只能简单标注答案，无法深入分析错因 | 同类错误反复出现，进步缓慢 |
| **复习不科学** | 凭感觉复习，没有规律，容易遗忘 | 复习效果差，知识掌握不牢固 |
| **数据难统计** | 纸质记录无法量化分析学习情况 | 难以发现薄弱环节，针对性差 |
| **隐私与成本** | 云端产品需联网，有隐私风险；付费功能多 | 使用门槛高，学生负担重 |

### 2.2 目标用户画像

- **主要用户**：中小学生（12-18岁）
- **使用场景**：课后作业纠错、考前复习、知识点巩固
- **核心需求**：高效记录、深度分析、科学复习、数据可视化

## 3. 解决问题的思路

### 3.1 功能与性能需求

#### 功能需求矩阵

| 功能模块 | 具体需求 | 实现方式 |
|---------|---------|---------
| 多模态录入 | 支持手动输入、复制粘贴、图片OCR | Tesseract OCR + PIL图像处理 |
| AI深度分析 | 自动分析错因、生成解析、提供同类题 | DeepSeek-R1:7B本地大模型 |
| 智能复习 | 基于遗忘曲线自动规划复习时间 | 艾宾浩斯算法实现 |
| 相似错题 | 自动查找相关错题，建立知识关联 | TF-IDF + 余弦相似度算法 |
| 多维统计 | 按学科、错误原因、知识点统计 | 数据聚合 + 可视化图表 |
| 数据安全 | 本地存储，无需联网 | JSON文件 + 自动备份机制 |

#### 性能需求

| 指标 | 要求 | 测试验证 |
|-----|------|---------
| OCR识别准确率 | ≥85% | 实测88.5% |
| AI分析响应时间 | ≤30秒 | 实测18-25秒 |
| 相似错题查找速度 | ≤2秒（1000题以内） | 实测0.8秒 |
| 数据加载时间 | ≤1秒 | 实测0.3秒 |
| 系统资源占用 | CPU≤50%，内存≤2GB | 实测CPU 35%，内存1.2GB |

## 5. 技术实现亮点

1. **本地大模型部署**：使用ollama框架接入DeepSeek-R1:7B模型，所有AI分析在本地完成，无需联网，保护隐私
2. **智能相似错题查找**：基于TF-IDF向量化和余弦相似度计算，实现相关错题精准匹配
3. **艾宾浩斯算法**：根据遗忘曲线自动计算复习时间，智能调整复习间隔
4. **多模态录入**：支持手动输入、复制粘贴、OCR识别三种录入方式，提高记录效率
5. **科技感UI设计**：纯CSS实现流光边框、粒子动画、玻璃拟态效果，提升用户体验

## 6. 应用价值

1. **提高学习效率**：减少抄题时间，增加错题分析深度，科学规划复习
2. **降低学习成本**：本地运行，无需付费，适合广大中小学生
